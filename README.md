# vest

A unit testing library for Godot.

## Features

* Define test suites
  * Both with `define()` and with class methods
* Assert library
* Parametrized tests
* Mocks
* Benchmarks
* Generate reports in TAP
* In-editor UI for running tests

## Compatibility

Godot v4.1.4 and up
